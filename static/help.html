How to Export Data from Interactive Brokers

Step-by-step guide to get your transaction data for UK tax calculations

Choose Your File Format

QFX Format (Recommended)

Quicken Financial Exchange format - most comprehensive data

✅ Pros:
  • Includes commission and fee data
  • More detailed transaction information
  • Better for accurate tax calculations
  • Includes currency exchange rates
⚠️ Cons:
  • Slightly more complex to export

CSV Format

Comma-separated values - simple and widely supported

✅ Pros:
  • Easy to export and view
  • Can be opened in Excel
  • Simple format
⚠️ Cons:
  • May lack commission details
  • Less comprehensive than QFX

Method 1: Export QFX File (Recommended)

1
Log into IBKR Client Portal

Go to https://www.interactivebrokers.com/portal and log in with your credentials.

Screenshot: IBKR Login Page
2
Navigate to Performance & Reports

Click on "Performance & Reports" in the main menu, then select "Statements".

Screenshot: Performance & Reports Menu
3
Select Activity Statements

Choose "Activity Statements" from the statements menu.

Screenshot: Activity Statements Selection
4
Configure Date Range

Set the date range for your tax year:

  • 2024-25 Tax Year: April 6, 2024 to April 5, 2025
  • 2023-24 Tax Year: April 6, 2023 to April 5, 2024
  • 2022-23 Tax Year: April 6, 2022 to April 5, 2023
Screenshot: Date Range Selection
5
Select QFX Format

Choose "QFX" as the file format. This format includes the most comprehensive transaction data including commissions and fees.

Screenshot: QFX Format Selection
6
Download the File

Click "Run" to generate the report, then download the QFX file when it's ready.

Screenshot: Download QFX File

Method 2: Export CSV File

1
Access Flex Queries

In IBKR Client Portal, go to "Performance & Reports" → "Flex Queries".

Screenshot: Flex Queries Menu
2
Create New Query

Click "Create" to create a new Flex Query for transaction data.

Screenshot: Create New Query
3
Configure Query Settings

Select the following sections:

  • ✅ Trades
  • ✅ Dividends
  • ✅ Corporate Actions
  • ✅ Cash Transactions
Screenshot: Query Configuration
4
Set Date Range and Format

Set your tax year date range and choose CSV as the output format.

Screenshot: Date Range and CSV Format
5
Run and Download

Save the query and run it to generate your CSV file.

Screenshot: Run Query and Download CSV

Important Notes

  • Tax Year Dates: UK tax year runs from April 6 to April 5 of the following year
  • File Size: Large files may take time to generate - be patient
  • Data Privacy: Your data is processed locally and not stored on our servers
  • Multiple Accounts: Export data separately for each IBKR account
  • Currency: The calculator handles multi-currency transactions automatically

Troubleshooting

Try exporting smaller date ranges (e.g., quarterly) and upload them separately to our calculator.

Use QFX format instead, or manually add commission costs in the results page after calculation.

Ensure your file covers the complete tax year and includes all transaction types. Contact support if issues persist.
