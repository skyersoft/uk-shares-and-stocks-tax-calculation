# How to Generate a QFX Report from Interactive Brokers

Based on the screenshots provided, here is a step-by-step guide to generating a QFX report from your Interactive Brokers account.

1.  **Navigate to Reports**:
    *   From the main menu, select **Performance & Reports**.
    *   From the dropdown, choose **Statements**.

2.  **Select Third-Party Downloads**:
    *   On the Statements page, look for the **Third-Party Downloads** section or tab and click on it.

3.  **Choose Quicken Format**:
    *   In the Third-Party Downloads section, you will see a list of available formats. Select **Quicken**.

4.  **Set the Reporting Period**:
    *   Choose the desired date range for your report. This is typically done using "From Date" and "To Date" fields.

5.  **Generate the Report**:
    *   Click the **Generate** button to create the QFX file for the selected period.

6.  **Download the QFX File**:
    *   After the report is generated, a **Download** button will appear. Click it to save the QFX file to your computer.
